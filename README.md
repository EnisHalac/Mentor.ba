# 🎓 Mentor.ba - Knowledge Exchange Platform
<img width="2024" height="777" alt="Logo" src="https://github.com/user-attachments/assets/b413e7af-58f3-48ca-b28a-4cc21ddd413d" />

> Connect with mentors, share knowledge, and build your career on the leading tutoring platform.

Mentor.ba is a comprehensive web application designed to connect tutors with students. The system offers advanced search capabilities, real-time communication, and learning history tracking, all supported by a dedicated administrative oversight dashboard.

## 🚀 Core Features

### 👥 Users and Roles
- **Unified User Model:** Every user can simultaneously act as both a student (receiving instructions) and a tutor (offering expertise).
- **Admin Role:** A dedicated dashboard for system-wide monitoring and management.

### 📚 Tutoring Listings (Ads)
Tutors can create detailed listings including:
- Title, description, and category (e.g., programming, languages, mathematics).
- Hourly rates.
- Delivery modes: **Online**, **On-site (Location-based)**, or **Hybrid**.
- **Advanced Filters:** Search by category, price range, location, work mode, and mentor reputation.

### ⭐ Ratings, Reviews & Reputation
- **Verified Reviews:** Only users who have completed a session with a specific mentor can leave a review.
- **Feedback System:** Includes 1-5 star ratings and detailed comments.
- **Automated Analytics:** Real-time calculation of average ratings and mentor reputation scores.

### 💬 Real-time Chat
- **Integrated Messaging:** Internal real-time chat system for session coordination and scheduling.
- **Transaction Proof:** Chat logs serve as evidence of communication and session completion.

### 📈 Learning History
Each user has access to their personal learning journey:
- Overview of topics studied and mentors consulted.
- Total hours completed.
- **Progress Tracking:** A solid foundation for enhancing personal CVs or professional portfolios.

### 🛠️ Administrative Dashboard
- **Management:** Full control over users, tutors, and active listings.
- **Verification:** Approval of professional licenses and certificates.
- **Conflict Resolution:** Reviewing reports, disputes, and detailed system analytics.

---

## 💻 Tech Stack

### 🎨 Frontend
- **React 19:** Core library for building dynamic user interfaces.
- **Vite:** Next-generation frontend tool for a fast development experience.
- **Tailwind CSS 4:** Utility-first framework for modern and responsive design.
- **React Router 7:** Advanced client-side routing and navigation.
- **Axios:** Reliable HTTP client for seamless API communication.

### ⚙️ Backend
- **Node.js & Express 5:** Robust server environment and API framework.
- **Prisma ORM:** Type-safe database mapping and management.
- **JSON Web Token (JWT):** Implementation of secure user authentication and authorization.
- **Bcrypt & Bcryptjs:** Advanced encryption for maximum password security.
- **Multer:** Middleware for handling image uploads (avatars and documentation).

### 🗄️ Database & Cloud
- **PostgreSQL:** Powerful relational database for reliable data persistence.
- **Supabase:** Backend-as-a-Service (BaaS) for database hosting and management.
